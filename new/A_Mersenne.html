Découverte des nombres de Mersenne

Définition

Les nombres de Mersenne sont de la forme : Mn=2n−1M_n = 2^n -1Mn​=2n−1, pour nnn un entier non nul.

Exemples

  • M1=21−1=1M_1 = 2^1 -1 = 1M1​=21−1=1, n'est pas premier, par définition
  • M2=22−1=3M_2 = 2^2 -1 = 3M2​=22−1=3, est premier, 2 aussi
  • M3=23−1=7M_3 = 2^3 -1 = 7M3​=23−1=7, est premier, 3 aussi
  • M4=24−1=15M_4 = 2^4 -1 = 15M4​=24−1=15, n'est pas premier, 3×53\times53×5
  • M5=25−1=31M_5 = 2^5 -1 = 31M5​=25−1=31, est premier, 5 aussi
  • M6=26−1=63M_6 = 2^6 -1 = 63M6​=26−1=63, n'est pas premier, 3×3×73\times3\times73×3×7
  • M7=27−1=127M_7 = 2^7 -1 = 127M7​=27−1=127, est premier, 7 aussi
  • M8=28−1=255M_8 = 2^8 -1 = 255M8​=28−1=255, n'est pas premier, 3×5×173\times5\times173×5×17
  • M9=29−1=511M_9 = 2^9 -1 = 511M9​=29−1=511, n'est pas premier, 7×737\times737×73
  • M10=210−1=1023M_{10} = 2^{10} -1 = 1023M10​=210−1=1023, n'est pas premier, 3×11×133\times11\times133×11×13
  • M11=211−1=2047M_{11} = 2^{11} -1 = 2047M11​=211−1=2047, n'est pas premier, 23×8923\times8923×89, pourtant 11 est premier
  • M12=212−1=4095M_{12} = 2^{12} -1 = 4095M12​=212−1=4095, n'est pas premier, 3×3×5×7×133\times3\times5\times7\times133×3×5×7×13

Le plus grand nombre premier connu a souvent été un nombre de Mersenne.

  • Nous présentons quelques aspects historiques, et ce sera l'occasion de découvrir le langage Python.

Aspect historique

Au XVIIe siècle

Au début du XVIIe siècle, Marin Mersenne, moine de l'ordre des Minimes, pense et démontre peut-être que « Si MnM_nMn​ est premier alors nnn aussi ». La réciproque est fausse, comme le montre n=11n = 11n=11.

Marin Mersenne fournit aussi une liste des nombres premiers « de Mersenne » jusqu’à l'exposant 257. Cette liste contenait des erreurs ; il y a de vraies difficultés, comme nous le verrons par la suite.

Au XVIIIe siècle

En 1772, Leonhard Euler prouve que M31M_{31}M31​ est un nombre premier. Il lui a suffit de poser 372372372 divisions pour cela.

Avec Python, on peut calculer M31M_{31}M31​ ainsi :

>>> 2**31 - 1
2147483647

Le nombre 2 147 483 6472\,147\,483\,6472147483647 est resté le plus grand nombre premier connu jusqu'en 1867.

Au XIXe siècle

Édouard Lucas a inventé un test de primalité, qui lui a permis de prouver en 1876 que M127M_{127}M127​ est premier.

Ce nombre a gardé le record jusqu'en 1951, le début de l'ère moderne des ordinateurs.

Au XXe siècle

En 1952, Raphael Robinson prouve que M521M_{521}M521​ est premier à l'aide d'un ordinateur. Depuis, le record est presque toujours détenu par un nombre de Mersenne.

Remarque
Python peut faire des calculs avec des entiers aussi grands que la mémoire de l'ordinateur le permet. Concrètement, Python peut travailler avec des nombres qui ont des millions de chiffres. En revanche, Python ne peut pas afficher de tels nombres. Ce n'est pas une histoire d'écran, c'est une histoire de conversion binaire vers décimal.

  • Avec un émulateur en ligne comme Basthon, on peut encore afficher M216 091M_{216\,091}M216091​ qui a 65 05065\,05065050 chiffres.
  • Le dernier record du XXe siècle, M6 972 593M_{6\,972\,593}M6972593​ peut encore être affiché raisonnablement avec Python installé sur poste fixe, il a 2 098 9602\,098\,9602098960 chiffres.

Au XXIe siècle

  • GIMPS est un projet de calcul partagé, fondé par George Woltman. Ce projet a permis de trouver les plus grands nombres premiers récents.
  • Les derniers records ont plus de 20 millions de chiffres, et sont difficiles à afficher avec Python.

Même si on ne peut pas afficher un nombre très grand, on peut travailler avec et, par exemple, calculer un résidu modulaire.
